Description list of HTML tags

Some HTML tags I have learnt

  • <th>
    Description: It defines the heading row of the table.
    Example: <th>This is the heading row.</th>

  • <a>
    Description: It specifies the hyperlink and the URL of the link. The href attribute inside the anchor element pair creates a hyperlink, it is also called hyperlink-reference attribute.
    Example: <a href="https://algonquincollege.com">Visit Algonquin College</a>

  • <img>
    Description: It uses to insert images of a website. The src attribute is used to define the source, i.e. the relative or absolute path of the image. The alt attribute represents the alternate text of the image.
    Example: <img scr="myself.jpg" alt="Image of myself">

  • <q>
    Description: It adds a small quotation in the text.
    Example: <q>This is a quotation.</q>

  • <strong>
    Description: It bolds the text to emphasis its importance.
    Example: <strong>This is a bold sentence.</strong>
